# Pokemon Battle Simulator

A turn-based battle system built in Python that simulates core mechanics of Pokemon-style combat.

## Features
- Turn-based battle system
- Type effectiveness system (Fire, Water, Grass)
- Attack and damage calculations
- Win/loss conditions
- Multiple Pokemon with different stats

## How It Works
The program models a battle between Pokemon using structured logic and functions.  
Each turn processes player actions, calculates damage, and updates game state until a winner is determined.

## What This Demonstrates
- Object-oriented programming concepts
- Conditional logic and control flow
- Managing game state across multiple turns
- Designing a larger multi-function program (~500 lines)

## How to Run
1. Open the file in Python
2. Run the program
3. Follow the on-screen prompts to play
